# Tab Maker 记谱语法

本项目把每首歌保存为一个 JSON 文件，并把其中的 `tabScript` 字段渲染成和弦谱。这个语法适合记录歌词、可选和弦，以及可选的数字音高。

## 歌曲 JSON

每个谱文件放在 `public/tabs/*.json`。

```json
{
  "title": "歌手 - 歌名",
  "originalKey": "C",
  "tabScript": "[I](1)今天,(2)我唱|[V](3)一次"
}
```

如果需要显示和弦图，可以额外加入 `chordDiagrams`。

## 基本排版

通常用一行表示一个段落或一句乐句组合。

```text
[I](1)今天,(2)我唱|[V](3)一次
[vi](5)下一,(4)句|[IV](3)这里
```

- 换行：开始一个新的显示段落。
- `|`：分隔小节。
- `,`：分隔小节内的歌词块。
- 分隔符前后的空格可以省略，也可以保留。

## 歌词块

一个歌词块的格式是：

```text
[和弦](音高)歌词
```

和弦与音高都可以省略，但歌词必须存在。

```text
[I](1)今天
[V]你好
(3)世界
只有歌词
```

歌词里可以使用 `/`，把同一个歌词块拆成多行显示。

```text
[I](1)今/天
```

## 和弦

和弦使用级数记法，应用会根据选择的调转成实际和弦名。

```text
[I]
[ii]
[V]
[vi]
```

大写级数显示为大三和弦，小写级数显示为小三和弦。

以 C 调为例：

```text
[I]  -> C
[ii] -> Dm
[V]  -> G
[vi] -> Am
```

支持的级数：

```text
I II III IV V VI VII
i ii iii iv v vi vii
```

### 升降号

在级数后面加 `b` 或 `#`。

```text
[VIIb]
[IV#]
```

### 和弦性质

在 `-` 后面加入和弦性质。

```text
[I-Maj7]
[ii-7]
[iii-7b5]
[V-sus4]
```

`-` 后面的文字会直接接在转调后的和弦名后面。

### 转位和分数和弦

使用 `/` 后接另一个级数来表示低音音。

```text
[V/VII]
[vi/III]
[iii-7b5/V]
```

`/` 后面的级数也会根据当前选择的调进行转换。

## 音高

音高使用数字简谱式的级数。

```text
(1)
(2)
(3)
(4)
(5)
(6)
(7)
```

可以用 `+1`、`+2`、`-1`、`-2` 标记高八度或低八度。

```text
(1+1)
(3+2)
(6-1)
(5-2)
```

音高默认隐藏，可以在应用中勾选“显示音高”来显示。

## 和弦图

和弦图是可选内容，写在歌曲 JSON 里，并按调分组。

```json
{
  "key": "C",
  "chords": [
    {
      "id": "CMaj9",
      "data": "CMaj9[x:3:2:4:3:x]"
    }
  ]
}
```

和弦图数据格式：

```text
名称[六弦:五弦:四弦:三弦:二弦:一弦]
```

每根弦的值可以是：

- `x`：不弹
- `0`：空弦
- 数字：品格位置

例子：

```text
Dm7[x:x:0:2:1:1]
```

## 完整示例

```json
{
  "title": "示例歌曲",
  "originalKey": "C",
  "chordDiagrams": [
    {
      "key": "C",
      "chords": [
        { "id": "CMaj9", "data": "CMaj9[x:3:2:4:3:x]" },
        { "id": "Dm7", "data": "Dm7[x:x:0:2:1:1]" }
      ]
    }
  ],
  "tabScript": "[I-Maj9](1)今/天,(2)我唱|[V-7](3)一次\n[vi](5)下一,(4)句|[IV](3)这里"
}
```
